na podstawie dostarczonego obrazka. Możesz wykorzystać bibliotekę: 
https://mvnrepository.com/artifact/net.sourceforge.tess4j/tess4j/4.4.1

 - Java8
 - Spring Boot
 - remote mySql

 - Swagger
 - tess4j: a Java JNA wrapper for Tesseract OCR API

 - Angular CLI
